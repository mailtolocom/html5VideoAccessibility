A) Description:

Le script HTML5VideoAccessibility est un addon pour les navigateurs web donnant plus de possibilité de manipulation des vidéo aux déficients visuels
Des raccourcis clavier spécifiques seront ajoutés, et un effort a été consenti pour forcer les vidéo à se lancer automatiquement au chargement des pages web.

B) Comment l'installer ?

1. vous devez d'abord installer l'extension greasemonkey (si vous avez mozilla firefox, ou tamper monkey si vous avez google chrome comme navigateur web.
Directement dans ces navigateurs, rendez-vous sur les pages listant les extensions qui leurs sont disponibles, et cliquez sur télécharger sur celles qui vous intéressent.

2. Décompressez le fichier HTML5VideoAccessibility.zip à un endroit de votre ordinateur.

3. Ouvrez avec le navigateur web souhaité le fichier "instal.html", et validez sur le lien installer le script.

4. Une page web de confirmation devrait alors apparaître. Vous y rechercherez le bouton ou lien de mise à jour ou d'installation du script, et vous validerez dessus.

C) Liste des raccourcis clavier de manipulation des vidéo

Lorsque vous êtes sur une page web contenant une vidéo, vous pouvez employer les raccourcis clavier suivants:

* insert+espace pour activer le mode formulaire qui est le mode dans lequel les raccourcis clavier s'exécuteront sans conflit.
Vous ferez la même combinaison de touche pour désactiver le mode formulaire.
Quel que soit le logiciel de revue d'écran, vous pouvez également échapper la prochaine combinaison de touches (Insert+3 pour jaws, insert+F2 pour NVDA) avant d'exécuter l'un des raccourcis ci-dessous.

* échap pour repositionner le focus sur l'objet vidéo ou ignorer la vidéo publicitaire lorsque vous êtes sur youtube

* shift+espace pour mettre pause ou relancer la vidéo sélectionnée

* maj+flèche gauche ou droite pour reculer ou avancer de 10 secondes

* Ctrl+flèche gauche ou droite pour reculer ou avancer de 60 
secondes

* Ctrl+maj+flèche gauche ou droite pour reculer ou avancer la vidéo de 
5 minutes;

* Maj+origine pour aller au début de la vidéo courante;

* Maj+fin pour aller à la fin de la vidéo courante;

* Maj+g pour appeler la boîte de dialogue "atteindre" par laquelle vous pourrez taper un temps de la vidéo à atteindre;

* Maj+flèche haut ou bas pour augmenter ou diminuer le volum de la video sélectionnéee

* ctrl+flèche haut ou bas pour basculer entre les différents objets 
vidéo sur la pages s'i il y en a plusieurs;

* r ou maj+r pour augmenter ou diminuer la vitesse de lecture de la vidéo courante;

* d pour faire dire les statuts de la vidéo courante, c'est-à-dire la position actuelle, la durée totale, le pourcentage de lecture, si en pause, si muet, le pourcentage de vitesse de lecture...

* m pour activer ou désactiver le muet;

* Maj+page suivante pour passer à la vidéo suivante sur youtube.

* u pour faire afficher l'URL de la vidéo sélectionnée

D) Quelques avertissement

* certains raccourcis clavier ne fonctionneront pas si vous les exécutez au dessus d'un champ de formulaire d'entrée de texte. Donc avant d'entrer en mode formulaire, ou déchapper la touche suivante, assurez-vous que vous êtes sur tout, sauf ce type de champ-là.

* sur certains site il est possible que maj+espace ne fonctionne pas pour lancer la lecture de la vidéo, que vous soyez obligés  de le faire par le bouton "play" qui est proposé sur la page. Mais ensuite, vous constaterez que les autres commandes fonctionnent.

* sur youtube, lorsque le lecteur passera à la vidéo suivante, le titre de la nouvelle vidéo ainsi que sa durée vous sera automatiquement lu par la synthèse vocale.


Enjoy !
